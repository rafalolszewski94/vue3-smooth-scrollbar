# vue3-smooth-scrollbar

Based on [vue-smooth-scrollbar](https://github.com/BlackBP/vue-smooth-scrollbar)

## Requirements
- Vue 3

## Project setup
```
npm run install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
